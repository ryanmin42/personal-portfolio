Flask
JavaScript
JQuery
HTML
CSS

VisualEyes is a smart home doorbell system designed to provide security & convenience for the blind & disabled within their own homes. Every time the doorbell rings, a webcam takes a picture of the person at the door and uses a facial recogntiion API, Kairos, to match them to one of two databases: trusted friends & family, or suspicious people. If the person is not found in either, they are marked as unknown. The matched database and name of the person at the door are communicated to the user through a voice interface system.

Intervue is a voice-enabled interviewing practice platform that offers sample questions and helpful feedback utilizing the Amazon Echo. Once a user response is recorded, it is sent over to the server and analyzed with IBM's Watson Tone Analyzer API, which returns 15 metrics based on speech patterns such as fear, confidence and excitement, and uses the metrics to deliver both positive and constructive feedback.

Pizza Party is an automated pizza deliery system that orders the perfect pizza suited to your cravings based on price, location and favourite toppings. The user's location is automatically detected using the Google Maps API, and they can enter their party size & 4 favourite toppings. Shopaholic

July 2017 - ongoing

HTML
CSS
JavaScript
Python
JQuery

Shopaholic is a Chrome extension that allows users to compare their online shopping options at the click a button. The user can highlight a word or phrase within a webpage, and the extension searches and scrapes data from different online retailers such as Amazon, eBay, and AliExpress to display product information that matches the highlighted word/phrase, such as product images, shipping location, price and features. The user can then choose to sort the results by price, relevance, or date posted.
